Algoritma & Pseudocode

Langkah-langkah lengkap implementasi metode Regula Falsi beserta flowchart dan kode.

1. Langkah-Langkah Algoritma

  1. Definisikan fungsi f(x)

    Masukkan persamaan f(x) = 0 yang akan dicari akarnya.

  2. Tentukan interval awal [a, b]

    Pilih a dan b sedemikian sehingga f(a) · f(b) < 0 (tanda berlawanan).

  3. Tentukan toleransi ε dan iterasi maksimum

    Misalnya ε = 0.0001 dan maks iterasi = 100.

  4. Hitung titik pendekatan xr

    Gunakan formula: xr = b − f(b)·(b−a) / (f(b)−f(a))

  5. Evaluasi f(xr)

    Hitung nilai fungsi pada titik xr yang baru ditemukan.

  6. Perbarui interval

    Jika f(a)·f(xr) < 0 → b = xr, jika tidak → a = xr

  7. Cek kriteria berhenti

    Jika |f(xr)| ≤ ε atau error relatif ≤ ε, hentikan. Jika tidak, kembali ke langkah 4.

  8. Output hasil

    Tampilkan xr sebagai aproksimasi akar beserta jumlah iterasi dan error.

3. Pseudocode

FUNCTION regulaFalsi(f, a, b, epsilon, maxIter):
    IF f(a) * f(b) >= 0 THEN
        RETURN "Error: f(a)·f(b) harus < 0"

    iter = 0
    xr_lama = a

    WHILE iter < maxIter DO
        fa = f(a)
        fb = f(b)

        // Hitung titik perpotongan sekanta dengan sumbu-x
        xr = b - (fb * (b - a)) / (fb - fa)
        fxr = f(xr)

        // Hitung error relatif (kecuali iterasi pertama)
        IF iter > 0 THEN
            error = |xr - xr_lama| / |xr| * 100   // dalam %

        // Cetak baris tabel iterasi
        PRINT(iter + 1, a, b, xr, fxr, error)

        // Cek kriteria berhenti
        IF |fxr| <= epsilon THEN
            BREAK

        // Perbarui interval
        IF f(a) * fxr < 0 THEN
            b = xr
        ELSE
            a = xr

        xr_lama = xr
        iter = iter + 1

    RETURN xr
END FUNCTION

4. Implementasi Python

# regulaFalsi.py
# Implementasi Metode Regula Falsi

def regula_falsi(f, a, b, epsilon=1e-6, max_iter=100):
    """
    Parameter:
        f        : fungsi f(x)
        a, b     : batas interval (f(a)*f(b) < 0)
        epsilon  : toleransi error
        max_iter : iterasi maksimum
    Return:
        dict berisi akar dan data iterasi
    """
    if f(a) * f(b) >= 0:
        raise ValueError("f(a)·f(b) harus bernilai negatif")

    results = []
    xr = a
    xr_prev = a

    for i in range(max_iter):
        fa, fb = f(a), f(b)
        xr_prev = xr

        # Formula Regula Falsi
        xr = b - (fb * (b - a)) / (fb - fa)
        fxr = f(xr)

        err_rel = None if i == 0 else abs((xr - xr_prev) / xr) * 100

        results.append({
            'iter': i + 1, 'a': a, 'b': b,
            'xr': xr, 'fxr': fxr, 'err_rel': err_rel
        })

        if abs(fxr) <= epsilon:
            break

        if f(a) * fxr < 0:
            b = xr
        else:
            a = xr

    return {'root': xr, 'iterations': results}


# Contoh pemakaian:
import math

f = lambda x: x**3 - 3*x - 1
hasil = regula_falsi(f, a=1, b=3, epsilon=1e-6)
print(f"Akar: {hasil['root']:.8f}")
